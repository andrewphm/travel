# Tavel mockup page

Mobile-first fully responsive website.

Features
- Custom framework 
- Fully responsive
- Side bar for tablet 
- Animations 
- Dark/Light mode
- Strong focus on using CSS variables and relative units
- Touch slider for mobile using SwiperJS (Effect coverflow style)
